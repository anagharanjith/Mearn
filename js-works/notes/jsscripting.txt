--------------------------------

- DOM :Document object model - tree representation of html elements in browser while loading a web page
-document object in js used to access DOM
- ways to applay js in html 
       - internal js : use script tag
       -external js  : define js code in external js file , to connect js file use script tag
-ways to access html elements in js
          -use tag names :document.getElementByTagName('tag-name)
          -use id :document.getElementByTagId  (id)
          -use class :document.getElementByTagClass  ('class-name)
          -use querySelector : document.querySelector('getElementByTagName/#id/.class-name')
          -use querySelectorAll : document.querySelectorAll('getElementByTagName/#id/.class-name')
