Document Object Model

Acessing DOM Elements

  1. use tag name : document.getElementByTagname('tag-name')
  2. use id :document.getElementByTagId (id)
  3. use class :document.getElementByTagClass ('class-name)
  4. use querySelector : document.querySelector('getElementByTagName/#id/.class-name')
  5. use querySelectorAll : document.querySelectorAll('getElementByTagName/#id/.class-name')
